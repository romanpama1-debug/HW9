## Reflection
The easiest part was displaying values on the webpage using JavaScript. The hardest part was understanding how Number conversion works with different types of strings. I learned that the Date object starts months at 0, so you have to adjust it. I also learned how Number() can turn invalid values into NaN. Finally, I learned how to update the DOM to show results instead of using console.log.
